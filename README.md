geotype
---

geotype is a cli tool for rendering geojson in your terminal.

![](https://dl.dropbox.com/s/pd6ewtiuazatwd8/Screenshot%202015-03-12%2000.39.18.png?dl=0)

##install

```sh
npm install geotype -g
```

##run

```sh
geotype vermont.geojson
```
